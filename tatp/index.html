Welcome to
Telecom Application Transaction Processing
Benchmark Suite
for Windows (64b)

TATP Benchmark Suite is a collection of software executing TATP Benchmark. TATP Benchmark is a mixed-load database benchmark for telecom applications. It generates a transaction-oriented workload, over several parallel client connections, targeted at a database system under test.

Note on the executables in this package: the TATP binaries have been build with the Windows ODBC driver manager. TATP execution requires aprropriate data source names (DSNs) to be defined in the driver manager (with the ODBC control panel).
